MAC 系统做的已经挺好了，但有些地方依旧不太完美，这也就是这些小工具出现的原因，让系统更加好用，更加强大！

3.1 搜索利器 Alfred2

一般性质的功能，10.10 以上使用Spotlight足以，如果你还没有开始使用，强烈建议开启 Spotlight 的快捷键，开启方法系统偏好设置-Spotlight-键盘快捷键-显示Spotlight搜索-调整你自己的就ok了，xirong 自定义的为 option+空格，关于Spotlight的使用技巧，请参考写的很棒的一篇文章OS X Yosemite 全新的 Spotlight 能做的 8 件事。

Spotlight 能做的事情，Alfred2 都能做，Alfred2 还能做一些Spotlight不能做到的事情，比如，一些常用的workflow，提升效率刚刚的，举个例子，当我想看自己的ip时候，两种方法：终端-ifconfig-找到ipv4地址 或者 系统偏好-网络-当前ip ，需要几步？ 如果使用 Alfred2，我只需要ctrl+空格，调出Alfred2搜索窗口，输入ip，即可展示我的ip地址，如下图：

想体验更多的workflow，直接去网站 http://www.alfredworkflow.com/ 上搜索你想要使用的下载安装即可，更多更全的 workflow 参考网站： http://www.packal.org/ 另：欢迎体验 xirong 写的几个wf：

  • 购物搜索 目前支持淘宝、天猫、京东、亚马逊，分别输入关键词tb、tm、jd、zcn 空格后加入想要搜索的词就ok，比如想要在京东搜索iPhone6 ，直接打开Alfred2，输入jd iPhone6 回车即可，打开京东的搜索结果。
  • 电子书搜索 xirong 经常阅读电子书，目前的效果排版价格都很公道，关键是省力气省空间，所以会经常去多看、京东、当当、亚马逊、豆瓣、百度等电子频道查找书籍，感到很麻烦，于是就有了这个workflow，打开Alfred2，输入ebook 明朝那些事 回车后会在浏览器中打开这几个网站的电子书搜索结果，你可以自己对比下从相应的网站购买即可。
  • Chrome History 以及 Chrome Bookmarks 可以用于搜索 Chrome 的收藏书签和历史记录（支持模糊搜索）
  • Evernote：支持印象笔记的全局搜索。这功能很好很强大，能够帮助自己快速定位到具体的印象笔记的条目之上。
  • Mail.app Search：支持 Email Apple 自带的邮件应用的全局搜索。
  • Airdrop：快速打开 Mac 的 AirDrop 功能。
  • Dash：代码 Doc 文档全局快速搜索。妈妈再也不用担心我查 API 的速度很慢了。
  • Sublime Text Projects：自动搜索索引范围下的 Sublime Projects。开发者提速工具， 这个个人非常喜欢，直接就载入项目文件了，速度非常之快，绝不拖泥带水。
  • Relaunched：重启没有响应的程序。
  • Show Desktop：一键无窗口，浑身变轻松。
  • TerminalFinder：Terminal 和 Finder 之间互相切换。

5.2 一键host切换

刚刚接触MAC，是不是还不知道怎么修改host内容？ 其实还是比较简单的，vim /etc/hosts 在打开的编辑器中添加你需要的内容即可。 但是，当你有很多的测试环境，需要不断的在各个测试环境中切换的时候，天哪，这样操作无法忍受啊！！Gas Mask 这款免费的host切换工具就出现了，使用简单，点击 Create 按钮可以创建本地（Local）、远程（Remote）、结合（Combined）三种类型的 hosts 文件。点击 Activate 按钮即可激活选中版本的 hosts 文件。除此之外，还可以点击系统菜单栏的 Gas Mask 图标进行切换，非常方便。github-下载地址
