关于time machine不可不知的技巧

发表于 2015-12-13 | 分类于 mac | 阅读次数

Apple 官网上关于如何使用 time machine 的介绍，请看使用 Time Machine 备份或恢复 Mac ，下面的几篇文章都是手把手具体的介绍每个部分怎么使用的，可以阅读。

  1. Time Machine 使用教程（一）：设置 Time Machine 备份你的 Mac
  2. Time Machine 使用教程（二）：在 Time Machine 磁盘创建不同分区
  3. Time Machine 使用教程（三）：为你的备份加把锁
  4. Time Machine 使用教程（四）：排除特定文件（夹）的备份
  5. Time Machine 使用教程（五）：如何查看、恢复特定文件
  6. Time Machine 使用教程（六）：如何手动删除 Time Machine 备份
  7. Time Machine 使用教程（七）：使用 Time Machine 备份将音乐数据迁移至新 Mac

推荐排除下面的文件夹，一般没有什么用处，还可以减少备份盘占用空间的大小

  1. 用户/资源库/Caches这个文件夹不备份 里面主要是一些缓存文件夹比如我的里面就主要是帮助 safari Google earth的缓存文件。
  2. private/var文件夹下的内容不备份，里面就是系统休眠时把内存条里的内容转存到硬盘上存放的地方。

Mac下数据备份策略

发表于 2015-12-12 | 分类于 mac | 阅读次数

事情如果有变坏的可能，不管这种可能性多么小，它总会发生（Anything that can go wrong will go wrong）。 - 墨菲定律（Murphy's law)

在此之前你有想过备份数据的事情吗？ 无论有或者没有，从此刻起，你都应该好好思考怎么去保护自己数据的可用性、安全性，以避免数据丢失、数据泄露所带来的时间经济等各种损失。我想能看到这篇文章的你，对 mac 或者 pc 的使用上基本不会有问题，无论你从事什么职业，工作生活中少不了与电脑打交道，那么自然的你会有很多数据，写的代码，安装的各种软件，系统的配置，处理的 word 文档、excel 表格、ppt 演讲稿等等，使用电脑的时间越长，这样子的数据就越多，产生的价值也就越大，尤其是随着你的身份的变化，这些数据的影响越来越重要。以前我一直侥幸的认为，在我身上不会发生数据丢失的问题，像硬盘损坏这种概率极低的事件更不可能发生，可万万没想到，上周升级了mac 上的 iTunes ，升级过程中重启导致再也无法开机，尝试各种办法无果，最终还是去的苹果零售店维修，万幸的是数据没有丢失。由于之前没有怎么关注，许多最近的数据都没有备份，维修工程师一直强调数据很可能丢失，即使找专业的恢复团队，花费一笔不小的费用不说，还不定能全部找回，当时我滴个小心脏啊……
